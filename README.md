# Application-de-gestion-a-rienne-en-Java-POO-
Application Java de gestion d'une compagnie aérienne permettant de gérer les passagers (ajout, modification, suppression, recherche) et les billets (réservation, annulation, remboursement). Développée avec une interface graphique Swing, une base de données PostgreSQL et le pattern DAO pour la persistance des données.
